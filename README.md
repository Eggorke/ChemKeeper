## English

### Overview

ChemKeeper is a Turtle WoW / Vanilla 1.12 addon for planning raid consumables and checking whether your character is stocked correctly before a raid.

It lets you build your own tracked consumables list, define target amounts, compare target vs bags vs bank, and withdraw missing items from your bank in a safe way.

### Features

- Draggable minimap button
- `Edit List` tab with your tracked consumables and target quantities
- Built-in consumable catalog based on the current RABuffs item list
- Custom item input by item link or item ID
- `Overview` tab with `Target`, `Bags`, `Bank`, and `Missing`
- `Withdraw Missing` button for safe bank withdrawal
- Bank-aware status messages in chat
- Saved list and window position

### How It Works

1. Add consumables from the catalog or through the custom item field.
2. Set the target amount for each tracked item.
3. Open your bank.
4. Switch to `Overview`.
5. Press `Withdraw Missing` to pull missing items from the bank.

### Notes

- The addon is designed to avoid unsafe stack-swapping behavior when moving items from the bank.
- If you do not have enough free bag space, ChemKeeper will warn you in chat.

## Русский

### Описание

ChemKeeper — это аддон для Turtle WoW / Vanilla 1.12, который помогает заранее собрать нужную химию и расходники перед рейдом.

Он позволяет собрать свой список расходников, задать желаемое количество, сравнить сколько нужно / сколько уже в сумках / сколько лежит в банке, а затем добрать недостающее из банка.

### Возможности

- Перетаскиваемая кнопка у миникарты
- Вкладка `Edit List` со списком отслеживаемых расходников
- Встроенный каталог расходников на основе актуального списка из RABuffs
- Добавление кастомного предмета по item link или item ID
- Вкладка `Overview` с колонками `Target`, `Bags`, `Bank`, `Missing`
- Кнопка `Withdraw Missing` для безопасного добора из банка
- Понятные сообщения в чат о статусе добора
- Сохранение списка и позиции окна

### Как использовать

1. Добавь расходники из каталога или через поле `Add Custom Item`.
2. Укажи нужное количество для каждого предмета.
3. Открой банк.
4. Переключись на вкладку `Overview`.
5. Нажми `Withdraw Missing`, чтобы аддон добрал недостающее.

### Примечания

- Аддон специально сделан так, чтобы не использовать рискованное поведение с заменой уже лежащих стаков при переносе из банка.
- Если в сумках не хватает свободного места, ChemKeeper напишет об этом в чат.
